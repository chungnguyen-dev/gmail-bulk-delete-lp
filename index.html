How it works

Up and running in 10 minutes

One-time setup. After that, just run one command and select your label.

1
Clone the repository
Download the tool from GitHub to your Mac.
git clone https://github.com/chungnguyen-dev/gmail-bulk-delete.git
2
Set up Python environment
Create an isolated Python environment using pyenv and venv, then install dependencies. Full guide in the README.
3
Get Gmail API credentials
Create a free Google Cloud project, enable the Gmail API, and download your credentials file. Takes about 5 minutes. Full guide in the README.
4
Run and authenticate
Run the script once. Your browser opens for Gmail authentication. Grant permission ‚Äî done! Token is saved for future runs.
python gmail_delete.py
5
Select label and confirm
The tool lists all your Gmail labels. Pick one, type DELETE to confirm, and watch 10,000+ emails disappear per minute.
